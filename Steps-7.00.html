Source Files to Download and Unzip the folder using SSIS package 7

Please check if you have 7 Zip software to unzip the zipped source folder using SSIS Package.

If you need to download the 7 zip software ... go to this below link to download the 7 Zip software to extract files from the Zipped Folder.

https://www.7-zip.org/


Click  and download the Zipped Folder 


Create new SSIS Project solution

1. Open Visual studio.

2. Click on File-> New -> Project.

3. Select Integration service project in new project window and give the appropriate name and location for project. And click ok.

4. The new project screen contains the following:

Tool Box on left side bar

Solution Explorer on upper right bar

Property Window on lower right bar

Control flow, data flow, event Handlers, Package Explorer in tab windows

Connection Manager Window in the bottom

5. Click on the Package1.dtsx under the Solution Explorer under SSIS packages and give the appropriate name.

6. In the SSIS Package the Drag Execute Process Task from Toolbox to Control Flow Pane and give the appropriate name. Double Click on Execute Process Task and then go to Process Tab.

Under the Process

 set Executable = Executable location (where 7z.exe file location)

Arguments = E + Zipped File Location (where you have saved your Zipped Folder)

Working Directory = Uncompressed folder location (destination Where you want to save the unzipped files)  and Click ok



7. Right Click on Execute Process Task and go to Properties. Set DelayValidation=True

8. After executing our SSIS Package we can see that the .zip file is uncompressed to our given folder.
